aidAR Platform

Platform Capabilities

Six integrated AI systems working in concert to deliver the most comprehensive medical intelligence platform available.

3D Anatomy Viewer

A fully interactive, procedurally generated 3D human body model rendered via Three.js WebGL. Supports CPR simulation, hemorrhage control, and airway management with GPU-accelerated geometry animations and OrbitControls.

Three.js · WebGL · TypeScript

Doctor AI Agent

A conversational multi-agent diagnostic engine powered by the Gemini LLM API. Interprets natural-language symptom descriptions, identifies critical conditions, and routes to evidence-based first-aid protocols in real time.

Gemini API · FastAPI · Python

Brain AI Agent

A neuro-imaging analysis agent that processes uploaded MRI and CT scans to detect anomalies, measure cortical thickness, and flag early markers of neurological conditions with granular confidence scoring.

Gemini Vision · Computer Vision · FastAPI

Skin AI Agent

A dermatological classification agent that evaluates uploaded skin lesion photographs, distinguishes between benign and malignant presentations, and delivers a risk score with a structured clinical recommendation.

Gemini Vision · Image Classification · FastAPI

Lung AI Agent

A respiratory health monitoring agent that analyzes chest X-rays to detect pneumonia, fibrosis, and obstruction patterns, delivering structured clinical findings with confidence intervals.

Gemini Vision · Radiology AI · FastAPI

Digital Twin Agent

A real-time biometric synchronization agent that maps physiological telemetry onto an interactive 3D patient model for continuous live health monitoring via WebSocket.

WebSocket · Three.js · Biometrics
aidAR Architecture

Technology Stack

The carefully chosen tools and frameworks powering the 3D viewer, multi-agent AI systems, and real-time biometric pipeline.

Three.js

WebGL 3D rendering engine powering the procedural anatomy viewer and digital twin visualization

Vite + TypeScript

Lightning-fast frontend build tooling with strict typing across all viewer, animation, and UI modules

Anime.js v4

Spring-physics animation timeline orchestrating all UI kinetics, procedural body animations, and background motion

Gemini API

Google Gemini LLM and Vision models driving all five AI agents — Doctor, Brain, Skin, Lung, and Digital Twin

FastAPI

High-performance Python async API backend orchestrating the multi-agent AI pipeline and biometric data streams

Python 3.11

Core language for backend AI logic, image processing pipelines, and biometric data analysis across all agents

WebSocket

Real-time bidirectional communication channel for Digital Twin biometric telemetry and live patient monitoring

Web Speech API

Browser-native TTS narration system providing real-time contextual voice guidance during 3D procedure training

CSS Custom Properties

Full adaptive theming engine managing light and dark modes with editorial design tokens across the entire platform
